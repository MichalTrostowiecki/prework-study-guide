HTML

  • Head element contains information the helps browser to render page correctly
  • Body element contains the content that user can see
  • Title element defines the title for a webpage
  • Meta element defines metadata
  • Link element connects an external resources to the HTML document
  • Script element embeds client-side scripts in the HTML document

CSS

  • CSS (Cascading Style Sheets) is a language which defines how webpage content looks
  • CSS defines style rules for specific elements of an HTML page and format them.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is a version control system that lets you manage and keep track of each version of your project
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • SSH : An SSH key is an access credential for the secure shell (SSH) network protocol
